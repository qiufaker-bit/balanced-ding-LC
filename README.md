# 平衡腚：山地平衡房车

横屏山地平衡载具小游戏 MVP。自习房车行驶在颠簸山路、陡坡、独木桥和悬空栈道上，玩家要像《登山赛车》一样按住油门/刹车控制车速，同时用倾斜、配重和咖啡道具守住舍友的魔性蹲姿。

## 试玩

- 本地服务预览：运行 `npm run serve` 后访问 `http://127.0.0.1:8787/web/index.html`
- 微信开发者工具：导入当前目录，项目类型选择小游戏，`AppID` 可先使用测试号或替换 `project.config.json` 里的 `touristappid`

## 当前实现

- 横屏 Canvas 游戏界面，`game.json` 已设置为 `landscape`
- 使用 `assets/rv-rider-character.png` 作为完整人物+凳子主角精灵，书本不再被裁掉
- 房车侧面改为半透明观察舱，人物模型不会被实心车身遮挡
- 房车和车厢内人物按半尺寸显示，轮胎不再巨大遮路
- 左下是油门，右下是刹车；不按油门时车辆会减速并停住
- 保留山地房车、双轮、车厢、车体俯仰、侧倾、颠簸和侧风
- 拖动车尾配重控制上坡/下坡时的车体俯仰
- 点击咖啡道具临时固定舍友脚部摩擦力
- 人物位置移到车厢中部，不再贴着车尾
- 以车厢底线为失败基准：臀部短暂脱离小白椅不算失败
- 椅子会先晃动，椅子带动人物晃动；人物倒到车厢底线、椅子倒到车厢底线，或人椅一起倒下才失败
- 人物和椅子更容易倾倒，短轴距更容易被陡坡和坑洼顶起
- 路面加入更尖的坡脊、深坑、跳坡和断桥弹跳
- 加入金币收集和绳索桥；绳桥路段会显著增加车体和椅子的抖动
